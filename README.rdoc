== README

Directions to load app:

1. Run `bundle`
2. Run `rake db:create`
3. Run `rake db:migrate`
4. Run `rake db:seed`
5. Run `rails server`

Visit 'localhost:3000' for the home page.  You can also use 'lvh.me:3000' domain instead, which will make it possible to add a subdomain to the url.  Visit 'tech.lvh.me:3000' to see the Tech job board that has been seeded.
